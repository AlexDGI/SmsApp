 Guía de subida de registros de SMS
  • El archivo que se va a subir debe ser en formato .XLSX
  • Debe tener las siguientes columnas con el valor que se requiera según la descripción de cada una:
    1. Telefono: El número de teléfono al que se enviará el mensaje (esta columna deberá tener formato de texto).
    2. Mensaje: El mensaje que se enviará.
    3. Dato: El dato/Deuda que está ligado al dueño del teléfono estará Dato1, Dato2 y Dato3
    4. Fecha de envío: La fecha en la que quiere que se realice el envío del mensaje (esta columna deberá tener formato de fecha latinoamérica).
    5. Hora de envío: la hora en la que quiere que se realice el envío (esta columna deberá tener formato de hora latinoamérica).
    6. Correo: El correo de la persona a la que llegará el mensaje (debe tener el formato correcto de correo).
  • Puede descargar la plantilla de Excel para facilitar su trabajo:
    • Plantilla Excel
  • Después de descargar la plantilla debe abrirla con Excel.
  • Agrega los datos en sus respectivas columnas.
  • Guarda y sube el archivo buscandolo en con el botón buscar que está arriba.
